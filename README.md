# Rock Paper Scissors — The Ultimate Edition
**Coded by:** Sehar | FAST NUCES Islamabad (BSAI)

## Features
- Colorful ASCII art for Rock, Paper, Scissors
- 3 AI difficulty levels — Easy, Medium, Hard
- Best of N rounds mode
- Win/loss streak system with combo messages
- Leaderboard saved to file (top 5 scores)
- Full match stats — win rate, favorite move, best streak
- No STL vector or algorithm used — pure C++ fundamentals

## Concepts used
File I/O, structs, arrays, functions, bubble sort,
ANSI color codes, game loop, AI logic
